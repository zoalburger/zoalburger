POS Kitchen Screen

Send the orders directly to the kitchen Screen in Odoo POS!

Odoo POS Kitchen Screen facilitates sending certain orders automatically to the kitchen. Moreover, the chef at the kitchen can approve or deny any order as per the availability. The cashier can check the status of the food being prepared in the kitchen anytime.

Save time and offer Quick Service

Customers prefer to go to the restaurants where they get quick services. So, now increase the speed of the food orders you get. Send Order details to the kitchen directly when the order confirms. The chef at the kitchen will be notified for the order. Then he/she can confirm the order and offer quick service.

Categorize the orders for the Kitchen

Not all the orders in the restaurant go to the kitchen. Suppose, the customer ordered any soft drink or packaged chips then transferring such orders to the kitchen is time wasting.

Odoo POS kitchen Display Screen facilitates you to segregate the categories that will go to the kitchen once order is placed or not.

Allow Specified POS Shop(s) to Send Order to the Kitchen

Do you have one or more POS shops in the same database and don't want to send orders to the kitchen from all POS shops? You can also set the specified POS shops from the backen so that only configured shop(s) can send the orders to the kitchen.

Detailed Features List


Add kitchen display Screen to Odoo POS

  • Show specified orders on Odoo POS Kitchen Screen Display.
  • Use Custom Generated Display To show kitchen orders on any screen.


Show Order Details on the POS Kitchen Screen

  • The Odoo POS Kitchen Screen shows Order Details to the Chef.
  • The chef will be notified for the order.
  • Once orders are confirmed by the chef; he/she can see them on grid view.


Launch the Odoo POS Kitchen Screen With a URL

  • No Physical Connection or POSBox is required to launch the secondary screen.
  • Use the custom created URl to launch the POS Kitchen Screen on any display monitor.


Separate Independent Kitchen Screen For Each POS Shop

  • A unique URL is created for each Odoo POS Shop.
  • A separate Odoo POS Kitchen Display screen can be launched for each POS Shop.
  • Each POS Shop Kitchen Display screen can be independent from each other.
